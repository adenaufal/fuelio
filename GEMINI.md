# Fuel Consumption Tracker PWA

This project is a Progressive Web App (PWA) for tracking motorcycle fuel consumption.

## Getting Started

All project requirements, technical specifications, and implementation details are outlined in the `start.md` file. Please refer to it for a comprehensive understanding of the project.
